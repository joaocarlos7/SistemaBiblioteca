📚 Sistema de Biblioteca
Sistema desenvolvido para gerenciamento de livros e empréstimos de uma biblioteca.

Foi desenvolvido como desafio de conclusão de um módulo de curso em Java na Rocketseat.

🚀 Funcionalidades
✅ Inserir livros novos
✅ Inserir dados do cliente
✅ Registrar empréstimos
✅ Salvamento persistente

📚 Stack / Tecnologias
Java / JDK 8+
Programação Orientada a Objetos (POO)
Encapsulamento
ArrayList
Filtros, busca e transformação
Java Time API
Streams API
Persistência JSON
